## 🔧 技术栈

### 前端
- Next.js 16 (App Router)
- TypeScript 5
- Tailwind CSS 4
- Radix UI (组件库)
- Lucide React (图标)
- Framer Motion (动画)
- Zustand (状态管理)

### 后端
- Supabase (PostgreSQL)
- Supabase Auth
- Supabase Realtime

### 部署
- Vercel (推荐)
- Docker

## 📝 使用指南

### 本地开发
```bash

npm run dev

# 构建生产版本
npm run build
```

```bash
# .env.local
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
SUPABASE_SERVICE_ROLE_KEY=your_service_role_key
```
