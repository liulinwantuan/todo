## 🔧 本地开发

### 环境要求
- Node.js 18+
- npm 9+

### 快速开始
```bash
# 安装依赖
npm install

# 配置环境变量
cp .env.local .env.local
# 填入 Supabase 配置

# 启动开发服务器
npm run dev

# 访问应用
http://localhost:3000 或 http://localhost:3009
```

### 数据库设置
```bash
# 1. 创建 Supabase 项目
# 2. 执行 src/lib/supabase-schema.sql
# 3. 获取 URL 和 API 密钥
# 4. 配置 .env.local
```
